# latin-to-hangul
Convert an ascii string to hangul unicode codepoint.
## Examples
```c
latin_to_hangul("SEO"); // 49436 (U+C11C)
latin_to_hangul("ul"); // 50872 (U+C6B8)
latin_to_hangul("kjefhjqhd"); // 0 (invalid)
```
